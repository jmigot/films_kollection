Comment ça marche?

En ouvrant FilmsKollection, vous arrivez sur la collection par défaut : Collection.
Vous pouvez commencer à y ajouter des films (Collection -> Scanner un dossier).
Une fois vos films ajoutés, vous pouvez lancer la mise à jour par internet (Collection -> Mettre les films à jour).
Quand la mise à jour est terminée, vérifiez rapidement votre collection, pour voir si certains films sont incomplets (en gras dans la liste), et corrigez les.
Vous pouvez maintenant profiter de votre collection !

Gérer les collections

Utilisez Fichier -> Créer une collection pour créer une nouvelle collection (créée dans le dossier collections à côté du programme) et Fichier -> Ouvrir une collection pour ouvrir une autre collection (située n'importe où dans l'ordinateur).

Ajouter/Supprimer des films

Utilisez Collection -> Ajouter des films pour ajouter des films (mettez un film par ligne), ou Collection -> Scanner un dossier pour scanner un dossier récursivement et trouver des films.

Utilisez le clic-droit sur un film ou une sélection de films, puis cliquez sur Supprimer pour les enlever de la collection. Si vous cochez la case Supprimer également les fichiers, le programme ira supprimer les fichiers correspondants s'ils existent).

Utilisez Collection -> Re-scanner les dossiers pour vérifier dans tous les dossiers déjà scannés si de nouveaux fichiers ont été ajoutés, ou si des fichiers correspondant à des films n'existent plus. Vous pouvez alors Ajouter ou Supprimer l'une ou l'autre des listes (ou les deux). Vous avez la possibilité de supprimer des films de chaque liste avant d'appuyer sur ces boutons en sélectionnant le ou les films voulus, puis en appuyant sur Suppr. ou Delete au clavier.

Remplir les informations des films

Utilisez Collection -> Mettre les films à jour pour remplir automatiquement les films avec internet. Si vous voulez modifier un film en particulier, un clic-droit sur le film permet plusieurs options :

  • Tous les choix pour avoir plusieurs choix de films pour un même titre, vous n'avez plus qu'à choisir le bon, vous pouvez même changer le titre et refaire la recherche,
  • Mettre à jour pour remplir automatiquement une fiche en allant chercher les informations sur internet grâce au titre,
  • Modifier pour modifier le film manuellement et remplir tous les champs soi-même;

Utiliser la recherche

Choisissez sur quel type de champ vous voulez rechercher (Recherche pour tous les champs).
Commencez à écrire votre recherche, des choix seront proposés au fur et à mesure.
Vous pouvez voir la liste des choix possibles en cliquant sur la flèche déroulante.
La recherche s'effectue en temps réel, vous voyez donc la liste de films rétrécir au fur et à mesure que vous écrivez.

  • Appuyer sur Entrée pendant que vous écrivez une recherche permet d'afficher le film suivant dans la liste,
  • Vous n'avez pas besoin d'écrire les accents (Destinee équivaut à Destinée),
  • La durée, l'année et la note peuvent être précédées de 5 signes différents :
    • ~ pour l'approximation (durée +/- 10%, note +/- 1, année +/- 2 ans)
    • <, <=, > et >= pour l'inférieur ou le supérieur

Afficher une sélection de films uniquement

Pour chaque film ou sélection que vous voulez garder, faites Clic-droit -> Cocher, ensuite cliquez sur Montrer les films : Cochés en haut à droite. Seuls les films qui vous intéressent apparaîtront alors.

Fusion/Défusion

En sélectionnant plusieurs films, il est possible de les fusionner (films en plusieurs parties, par exemple). La fusion fera disparaitre tous les films de la liste excepté le premier de la sélection. La fiche du premier film proposera un menu pour les boutons Lancer le film et Lancer la bande-annonce pour choisir de quel film parmis tous ceux fusionnés on parle.

Défusionner un film fera réapparaître dans la liste tous les films fusionnés avec ce film.

Changer dossier

Si vous décidez de renommer un dossier comportant plusieurs films, les films de votre collection ne trouveront plus leur fichier associé. Pour remédier à ceci, faites une recherche par dossier en sélectionnant celui que vous venez de renommer, puis sélectionnez tous les films présents dans la liste (Ctrl + A) et faite clic-droit, puis Changer le dossier pour leur réaffecter un nouveau chemin. Vous pouvez trouver l'ancien dossier dans la liste déroulante, et vous n'avez plus qu'à faire le changement de nom voulu.

Renommer les fichier

Si vous souhaitez renommer tous vos fichiers correctement, vous pouvez lancer Collection -> Renommer les fichiers et choisir quel champ utiliser (le titre ou le titre original).
Parfois le titre original est vide, c'est pourquoi le programme vous propose de le remplacer par le titre dans ces cas-là.
Sur les systèmes Windows, certains caractères sont interdits dans les noms de fichiers, le programme va alors les remplacer par des caractères valides ou les supprimer (selon les caractères). Si vous êtes sur un autre système, vous pouvez décocher la case, mais nous vous conseillons de la garder quelque soit votre système (ainsi si vous voulez accéder au fichier sur un autre système basé sur Windows, vous n'aurez pas de problèmes).

Préférences

  • Collection par défaut définit quelle collection sera ouverte au démarrage du programme. La liste des collections possibles est créée à partir des fichiers du dossier collections à côté du programme,
  • Nombre d'acteurs à afficher définit combien d'acteurs le programme doit afficher dans les infos d'un film,
  • Taille minimum de film pour le scan (en Mo) définit la taille minimale que devra faire un fichier pour être pris en compte en tant que film lors du scan d'un dossier,
  • Montrer le chemin complet des fichiers définit si le programme doit afficher l'url complète d'un film, incluant les dossiers le contenant, ou simplement le nom du fichier avec son extension,
  • Lancer directement les films aléatoires définit si le programme doit lancer les films choisis avec le bouton Film aléatoire de la barre de recherche ou simplement afficher les infos du film et attendre que l'utilisateur appuie lui-même sur le bouton Lancer le film,
  • Extensions à prendre en compte pour le scan de dossiers définit la liste des extensions (une par ligne) qui doivent être considérées comme étant des films lors du scan de dossiers (ne peut être composé que de lettres minuscules/majuscules, de chiffres et de - et _);

Explorer la collection

FilmsKollection utilise un format de base de données nommé SQLite (version 3), et toutes les données d'une collection sont stockées dans un fichier Nom-de-la-collection.db3. Pour lire ce fichier vous-même, vous devez utiliser un logiciel externe. Nous vous proposons un logiciel libre et gratuit : SQLite database browser.

  • Site web officiel : http://sqlitebrowser.sourceforge.net
  • Téléchargement : http://sourceforge.net/projects/sqlitebrowser/

Astuces

Faire un double clic sur un film dans la liste permet de lancer le film.

Mettre le logiciel à jour

Si le téléchargement automatique ne fonctionne pas, réessayer en lançant FilmsKollection avec des droits administrateurs.
Si vous n'arrivez toujours pas à mettre à jour, vous devrez télécharger vous-même la dernière version sur le site web (http://www.filmskollection.org). Une fois décompréssée, copiez-collez votre dossier collections et votre fichier Config.cfg dans le nouveau dossier pour récupérer vos collections et votre configuration.
