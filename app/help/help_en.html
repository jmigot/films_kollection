How does it work ?

By opening FilmsKollection, you are managing default collection: Collection.
You can start to add films (Collection -> Scan a dir).
As soon as your films are added, you can launch the internet update (Collection -> Update films).
When update is finished, check rapidly your collection, to verify if some films are uncomplete (bold in the list), and correct them.
You can now enjoy your collection !

Manage collections

Use File -> Create collection to create a new collection (created in the collections dir, close to program) and File -> Open collection to open an other collection (located anywhere in the computer).

Add/Remove films

Use Collection -> Add films to add films (put one film per line), or Collection -> Scan a dir to scan a dir recursively and find films.

Use right-clic on a film or a films selection, then clic on Remove to remove them from collection. If you check Also delete files box, program will delete files corresponding if they exist).

Use Collection -> Re-scan dirs to check in all dirs already scanned if new files have been added, or if files corresponding to some films don't exist anymore. Then you can Add ou Remove one or other of lists (or both). You have the possibility to remove films of each list before clicking on buttons by selecting film(s) you want, then pushing Suppr. or Delete on keyborad.

Fill films informations

Use Collection -> Update films to fill films automatically via internet. If you want to modify a particular film, a right-clic on the film allows several options :

  • All choices to see several choices of film for a same title, you just have to choose the right one, you can even switch title and re-do the search,
  • Update to fill a sheet by getting informations on the internet using title,
  • Modify to modify manually the film and to fill all fields yourself;

Use search engine

Choose which kind of field you want to search on (Search for all fields).
Start to write your search, some choices will be proposed to you.
You can see possible choices list by clicking on arrow down.
Search is done in real time, you can see list of films reducing as you are writing.

  • Push Return while you are writing a search permit to show the next film in the list,
  • You don't need to write accents (Destinee is the same as Destinée),
  • Duration, year and note can be preceded by 5 different signs:
    • ~ for approximation (duration +/- 10%, note +/- 1, year +/- 2 years)
    • <, <=, > et >= for inferior or superior

Show a selection of films only

Foreach film or selection you want to keep, do Right-clic -> Check, then click on Show films: Checked in top right of window. Only films that your are interested in will be shown.

Fusion/Unfusion

By selecting several films, it is possible to fusion them (films cut in some parts, for example). Fusion will make disappear of list all films except the first one of selection. The sheet of the first film will propose a menu on buttons Launch film and Launch trailer to choose which film among those fusioned we want to launch.

Unfusion a film will make reappear in the list all fusioned films with this film.

Switch dir path

If you decide to rename a dir containing some films, films of your collection won't find their assoced file anymore. To remedy to this, do a search by dir by seleting that one you just renamed, then select all films present in the list (Ctrl + A) and do Right-clic, then Switch dir path to reassign them a new dir path. You can find the old dir path in the arrow down, and you just have to change the name as wanted.

Rename files

If you wich to rename all your files correctly, you can launch Collection -> Rename files and choose which field to use (title or original title).
Sometimes original title is empty, so program propose to replace it with title in these cases.
On Windows systems, some charaters are forbidden in filenames, program will so replace them with valid characters or remove them (depending on characters). If you are on another system, you can uncheck the box, but we advice you to don't do this (so if you want to access your files on another Windows-based system, you won't have any problem).

Preferences

  • Collection by default define which collection will be opened at start of program. List of all possible collections is created from files of the collections dir, close to program,
  • Number of actors to display define how many actors the program must show in film informations,
  • Minimum file size for scan (Mo) define the minimum size that a file must have to being taken as a film during scan of a dir,
  • Show full path of files define if program must show the complete url of a film, including dirs containing it, or simply show the filename with its extension,
  • Launch directly random film define if program must launch directly films chosen with button Random film of search bar or simply show informations of a film and wait for user to click himself on Launch film button,
  • Extensions to take when scanning dirs define list of extensions (one per line) that must be considered as films when scanning dirs (can only be composed of letters lowercase/uppercase, numbers and - and _);

Browse collection

FilmsKollection use a database format called SQLite (version 3), and all collection data are stored in a file Name-of-collection.db3. To read this file yourself, you must use an external program. We propose you an open and free program: SQLite database browser.

  • Official web site: http://sqlitebrowser.sourceforge.net
  • Download: http://sourceforge.net/projects/sqlitebrowser/

Tricks

Doing a double-click on a film in the list launchs the film.

Update program

If automatic download doesn't work, retry by launching FilmsKollection with administrator rigths.
If you still can't update, you have to download yourself the last version on the web site (http://www.filmskollection.org). Once uncompressed, copy-paste your collections dir and your Config.cfg file in your new dir to get your collection and configuration back.
